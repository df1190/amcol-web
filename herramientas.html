Rojo = por debajo de tmin  ·  Amarillo = margen < 1 mm  ·  Verde = sobre tmin

API 653 §4.3.2 — Tasa de corrosión: CST = (tprev − tact) / intervalo. CLT = (tnom − tact) / años desde construcción. Se usa la mayor como tasa de diseño.

API 653 §4.3.3 — Vida remanente: RL = (tact − tmin) / Cdiseño. Intervalo máximo siguiente = MIN(RL / 2, 10 años) según §6.4.

tmin de referencia: Fondo general §4.4.4 → MAX(2.5 mm, tnom × 0.50). Anillo anular → 6 mm (si G > 1.0). Cuerpo → calculado por presión hidrostática.

Esta calculadora es de apoyo — el inspector API 653 certificado es responsable del criterio final.
